# NavAI Operations Suite - Backend

## Overview

The NavAI Operations Suite is an enterprise application designed to streamline operations and enhance productivity. This document provides an overview of the backend setup and usage.

## Tech Stack

- **Backend Framework**: FastAPI
- **Database**: PostgreSQL
- **ORM**: SQLAlchemy
- **Authentication**: JWT
- **Data Validation**: Pydantic

## Folder Structure

The backend is organized into a feature-based architecture:

```
backend/
├── app/
│   ├── auth/          # Authentication-related functionalities
│   ├── core/          # Core functionalities and configurations
│   ├── database/      # Database connection and SQLAlchemy setup
│   ├── middleware/     # Middleware components
│   ├── models/        # SQLAlchemy models for database tables
│   ├── routers/       # FastAPI routers for API endpoints
│   ├── schemas/       # Pydantic schemas for data validation
│   ├── services/      # Business logic and service layer functions
│   └── utils/         # Utility functions
├── requirements.txt    # List of dependencies
├── .env.example        # Template for environment variables
└── README.md           # Documentation for backend setup and usage
```

## Setup Instructions

1. **Create a Python Virtual Environment**:
   - Run the following command to create a virtual environment:
     ```
     python -m venv venv
     ```

2. **Activate the Virtual Environment**:
   - For Windows:
     ```
     venv\Scripts\activate
     ```
   - For Linux:
     ```
     source venv/bin/activate
     ```

3. **Install Dependencies**:
   - Install the required packages listed in `requirements.txt`:
     ```
     pip install -r requirements.txt
     ```

4. **Database Configuration**:
   - Update the `.env` file with your PostgreSQL connection details.

5. **Run the Application**:
   - Start the FastAPI application:
     ```
     uvicorn app.main:app --reload
     ```

## API Documentation

The API documentation is automatically generated by FastAPI and can be accessed at `http://localhost:8000/docs` after running the application.
